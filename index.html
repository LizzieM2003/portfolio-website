Some blurb about me

Hello, I'm Liz. I have approximately 20 years experience of designing, developing and maintaining Oracle relational database systems. However my real interest is in developing websites and web apps, so last year I gave up my job so I could devote my time to studying web development. Having a masters degree in IT and a computing background I felt confident that I would be successful with a course of self study. I started with a number of courses on Udemy, edX and egghead.io covering HTML5, CSS3 and JavaScript. It surprised me how much there was to learn with just these three subject areas alone and I still discover something new every day.

In order to make myself a more appealing prospect for employers I also decided that I should become familiar with at least one or two JavaScript frameworks as well as server side technologies. I attended a number of JavaScript meetups to find out more from the JavaScript community about the popular frameworks currently out there and after dabbling a bit with Angular 2 I settled on React. React's API is extremely simple to use compared with Angular 2's and there is such a rich ecosystem of libraries that you can use with React. I particularly enjoyed incorporating Redux into the applications I built with React - the concept of a centralised application state updated by reducers consuming actions made so much sense to me.
